# Quant Stock Analysis Project
Analyzing historical stock correlations using Python (yfinance, pandas) for quant research exploration.

## Tools Used
- Python (Anaconda)
- Jupyter (Notebook)
- yfinance (for data)
- pandas & numpy (for cleaning & correlation)
- matplotlib
- Tableau (for data visualisation)

## What's Inside
- 'Quant_Research_Project.ipynb': core code notebook
- 'Stock_Correlation_Visual.pdf': Visual for results

## Dataset
Data pulled using 'yfinance' for:
- Apple (AAPL)
- Microsoft (MSFT)
- Google (GOOGL)
- Amazon (AMZN)

Timeframe: 2022-2023

## Future Plans
- Expand to include moving averages or volatility measures
- Build trading signals using historical correlations
